# Contributing

## Deploy
Static site on GitHub Pages. `git push origin main` triggers a rebuild (~1 min). Verify a live
change by checking the actual CSS/copy through Cloudflare — response size/caching can mask a stale
deploy.

## Legal pages
`agb`/`datenschutz`/`impressum` (DE) each have an `-en` English twin — keep both in sync when either
changes.

## Brand assets
`brand/` is the single canonical home (`BRAND.md` + `Bonvio Brand Book.html` + logos). Don't create
copies elsewhere.

## Copy changes
Product/pricing facts here must match the live product (`receipt-mvp` repo) — check before changing
rates, tier names, or feature claims.
